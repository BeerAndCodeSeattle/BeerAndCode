Welcome to Beer && Code Seattle

Beer && Code is a bi–weekly Seattle meetup. Participants meet at least one of these criteria:

  • Uses any programming language for work or for fun
  • Enjoys having a tasty drink while using aforementioned programming language
  • Likes the idea of meeting other Beer && Coders and building community
  • Thinks that being challenged by and learning from others is a great idea

Sound cool so far? Head over to the Calendar and come to our next meetup.

If you're new, here are some common questions you might have:

  • I'm just learning programming and I'm a noob. Should I come? Yes!
  • I'm actually a designer and I use BLAH BLAH BLAH. Should I come? Yes!
  • I don't have anything to work on. Should I come? Yes! (we'll find you something)

At the risk of sounding repetetive, Beer && Code is for anyone.

We meet at 7 pm and our location changes every few months.

We like beer. We like code. We like people. Come hang out with us.
